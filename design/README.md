# 🎨 Trading Signals App — Premium Design

## Как импортировать в Figma

### Способ 1: HTML to Figma (Рекомендуемый)
1. Открой `design.html` в браузере Chrome
2. В Figma установи плагин **"html.to" by Builder.io** (бесплатный)
3. Установи расширение Chrome **"html.to.design"** от Builder.io  
4. Открой страницу `design.html` в Chrome
5. Нажми на расширение → Import to Figma
6. Все экраны будут импортированы как Figma frames

### Способ 2: Скриншоты + Auto Layout
1. Открой `design.html` в браузере
2. Каждый экран — отдельный iPhone frame (390×844)
3. Сделай скриншоты каждого экрана
4. Импортируй в Figma и используй как reference

### Способ 3: SVG Export
1. Открой `design-figma-export.svg` — это SVG версия всех экранов
2. Импортируй в Figma: File → Import → выбери SVG файл
3. Все элементы будут как векторные объекты
